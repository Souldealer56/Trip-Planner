# Plan: Phase 9 - RSVP Nudging & Roster Tracking

## User Review Required

> [!IMPORTANT]
> This phase requires executing a database schema migration to add `notes` to `rsvps` table and `last_rsvp_nudge_at` to `trips` table. Please run the following SQL command in your Supabase dashboard SQL Editor before starting the execution:
> ```sql
> ALTER TABLE rsvps ADD COLUMN notes TEXT;
> ALTER TABLE trips ADD COLUMN last_rsvp_nudge_at TIMESTAMP WITH TIME ZONE;
> ```
