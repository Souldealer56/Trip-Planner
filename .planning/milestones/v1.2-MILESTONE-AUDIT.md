## Cross-Phase Integration & Flows

- ** Roster-Split Integration:** Roster queries committed participants correctly to populate the check field split selectors during expense logging.
- ** Payer-Ledger Joins:** Expenses table joins with the users table to retrieve the payer's first name, which populates both the Ledger and Settle lists.
- ** Solver-Conversion Flows:** Logged expenses are converted dynamically from their input currency to the trip's base currency using live rates before feeding balances to the greedy matching debtor-creditor transaction resolver.
