## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(220, 25%, 8%)` | Page backgrounds (`--bg-main`) |
| Secondary (30%) | `hsl(220, 22%, 12%)` | Input backgrounds, cards (`--bg-surface`, `--glass-bg`) |
| Accent (10%) | `hsl(220, 85%, 57%)` | Primary button bg, active status, input focus rings (`--primary`) |
| Success Accent | `hsl(160, 80%, 45%)` | Verification complete badge, success checkmark (`--accent`) |
| Destructive | `hsl(0, 80%, 60%)` | Error notifications, disconnect confirmation |

Accent reserved for: Primary action buttons (CTAs), focused input borders, and success status indicators.
